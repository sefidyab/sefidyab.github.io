# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-02

### Added
- Initial release of Sefid Yab
- Timeline Scanner - automatic user extraction
- Profile Forensics - location, device, ID, creation date
- Anomaly Detector - White SIM detection
- Auto Tracker - automatic tracking
- Database Sync - localStorage + GitHub sync
- Bilingual support (Persian/English)
- Dashboard with search and filter
- Export to CSV/JSON
- Batch processing
- SEO-optimized website
